# AI Prompt Pack — 20 Premium Prompts

## Coding Prompts

**Code Reviewer:** "Review the following code. Identify: (1) security vulnerabilities, (2) performance issues, (3) anti-patterns, (4) best practice violations. Rate 1-10 and provide specific fixes.\n\n[CODE]"

**Bug Hunter:** "Debug this code methodically. List every possible bug, explain why each is a problem, provide a minimal failing test case, then suggest fixes ranked by priority.\n\n[CODE]"

**Architecture Advisor:** "I'm building [SYSTEM]. Propose an architecture: choose technologies with reasoning, identify bottlenecks, suggest scaling strategies. Include trade-offs.\n\n[REQUIREMENTS]"

**Test Generator:** "Write comprehensive tests for this code: happy path, edge cases, error handling, and mocks. Use [FRAMEWORK].\n\n[CODE]"

**Code Explainer:** "Explain this code to a junior developer. Break it down line by line, explain the purpose, and describe any patterns used.\n\n[CODE]"

## Writing & Marketing Prompts

**SEO Blog Post:** "Write a 1500-word SEO blog post on [TOPIC]. Include: compelling H1, meta description (under 160 chars), 4+ H2 sections, actionable advice, conclusion with CTA. Target keyword: [KEYWORD]"

**Email Subject Lines:** "Write 10 email subject lines for [EMAIL PURPOSE]. Mix curiosity, urgency, and benefit-driven angles. Target audience: [AUDIENCE]"

**Landing Page Copy:** "Write landing page copy for [PRODUCT]. Structure: hero headline, problem statement, solution, 3 key benefits, social proof placeholder, CTA. Tone: [TONE]"

**Product Description:** "Write a compelling product description for [PRODUCT]. Lead with the biggest benefit, address top 3 objections, include a FAQ section."

**Social Media Calendar:** "Create a 2-week social media content calendar for [BRAND/NICHE]. Include: platform, post type, caption with hook, hashtags, and posting time."

## Productivity Prompts

**Meeting Summariser:** "Summarise this meeting transcript. Extract: key decisions made, action items with owners, open questions, and next steps.\n\n[TRANSCRIPT]"

**Task Decomposer:** "Break this goal into specific, actionable tasks: [GOAL]. For each task include: description, time estimate, dependencies, and success criteria."

**Decision Framework:** "I need to decide between [OPTION A] and [OPTION B]. Build a decision matrix: criteria, weights, scores, recommendation. Context: [CONTEXT]"

**Weekly Planner:** "Create a focused weekly plan. I have [N] hours. My top goal is [GOAL]. My constraints are [CONSTRAINTS]. Prioritise ruthlessly."

**Meeting Agenda:** "Create a tight 45-minute meeting agenda for [TOPIC]. Include: pre-read materials, timed sections, decision points, and desired outcome."

## Creative Prompts

**Brand Namer:** "Generate 20 brand names for [BUSINESS TYPE]. Requirements: [REQUIREMENTS]. Include: domain availability considerations, tone notes, and top 3 recommendations."

**Tagline Generator:** "Write 15 taglines for [BRAND/PRODUCT]. Mix: benefit-focused, clever, bold, and minimalist styles. Target: [AUDIENCE]"

**Story Hook:** "Write 5 compelling opening paragraphs for a story about [TOPIC]. Each should use a different hook: action, dialogue, description, question, or statement."

**Presentation Opener:** "Write 3 powerful opening lines for a presentation on [TOPIC] to [AUDIENCE]. Each should immediately establish credibility and create curiosity."

**Cold Outreach:** "Write a cold outreach message for [PURPOSE]. Keep it under 80 words. Lead with value, not ask. Target: [RECIPIENT ROLE] at [COMPANY TYPE]."

Copy any prompt above into ChatGPT, Claude, or your preferred AI. Replace the [PLACEHOLDERS] with your specifics. Iterate and refine.
