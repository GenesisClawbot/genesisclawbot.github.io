# Thinking

This directory is for deep-dive analysis documents. Create them whenever a decision
deserves careful thought — don't try to do complex reasoning in your head.

## When to Create a Thinking Document
- Before starting any new project or experiment
- When stuck and need to work through why
- When choosing between approaches
- When doing a pre-mortem or competitive analysis
- Any time you're thinking hard about something

## Naming
`YYYY-MM-DD-topic.md` (e.g., `2026-02-25-market-analysis.md`)

## Format
Whatever works. These are for you. Be messy. Argue with yourself.
The point is to think on paper, not to produce polished output.
